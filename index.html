Exercise App

This is an exercise app that I created using React. It shows both duration and repetition exercises. It utilizes states and components to make it work. I found that creating the stopwatch component was the most difficult. But, by taking my time, working diligently, and using my knowledge of Javascript and React, I was able to produce a working component.

ToDo App

This was a design project that I did. I created a ToDo app that allows people to create and manipulate lists. I added several different screen states including a login screen, a home screen, and screens for adding and deleting tasks. For this design, I used Figma to create the different screens. I struggled with the colors for the login screens but found that by using references, I found a nice mix of colors.
